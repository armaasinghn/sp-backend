# Security Pass — User Guide

## Admin

1. **Log in** as `admin@securitypass.local` / `password`
2. **Dashboard** — view all 4 KPI tiles (Total Passes, Pending, Approved Today, Total Visitors)
3. **Passes** — full list with filters by status, search by name; click any row to view detail
4. **Request Pass** — submit a new pass on behalf of a visitor
5. **Visitors** — view directory of all registered visitors; click to see visit history
6. **Notifications** — view all system alerts; click any to jump to the relevant pass
7. **Admin** — user management: view all staff accounts, roles, departments
8. **Profile** — update name/email/phone, change password, upload photo

---

## Approver

1. **Log in** as `priya@company.com` / `priya@123` (or any other approver)
2. **Dashboard** — scoped to your assigned passes only
3. **Pending Actions** — approve or reject passes directly from the dashboard card
4. **Passes** — your passes; filter by status
5. **Pass Detail** — full visitor info, approve/reject buttons, QR code on approval
6. **Notifications** — alerts when new passes are assigned to you

---

## Gate User

1. **Log in** as `ravi@company.com` / `ravi@123`
2. **Passes** — all passes; use Entry / Exit buttons to log gate activity
3. **Visitors** — visitor directory with entry status
4. **Notifications** — gate alerts

---

## Visitor

1. **Log in** as `visitor@securitypass.local` / `password`  
   *or* click **Register as Visitor** on the login page to self-register
2. **Request Pass** — fill visitor details, select meeting officer and purpose, set visit dates
3. **My Passes** — view all your submitted passes and their status
4. **Pass Detail** — on approval, view QR code, download or print your gate pass
5. **Profile** — update your details and password

---

## Account Recovery

- Click **Forgot?** on the login page
- Choose **Forgot Password** to reset via a 3-step OTP flow
- Choose **Forgot User ID** to recover your email by entering your mobile number or full name

---

## Quick Test Login

Use the **"Quick Test Login"** dropdown on the login page to instantly fill any account's credentials — useful for testing all roles without typing.
